1. Seat planning are based on first come, first served basis.
2. Customer who came first deserves the best sit planning, starting from the middle row where best screen view is possible and then closer.
3. If customer requests number of  seats more than the number of seats available in a row, place maximum number of available
seats on that row and manage seats for remaining customers in the rows nearest best place (closer to middle row). (Maximize theater Utilization)
4. Theater is unable to reserve seat if the number of  seats required is greater than number of available seats. //
5. For safety buffer of 3 seats between a group reservation or individual reservation.
